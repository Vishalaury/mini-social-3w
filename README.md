# Mini Social App (3W Internship)

 <!-- Overview -->
A mini social media app where users can signup, login, post text or images, like and comment.

 <!-- Tech Stack -->
- Frontend: React.js
- Backend: Node.js + Express
- Database: MongoDB Atlas
- Deployment:
  - Frontend → Vercel
  - Backend → Render

 <!-- Features -->
- User Authentication (Signup/Login)
- Create Post (Text/Image)
- View Feed (All users’ posts)
- Like & Comment System
- Responsive Design
