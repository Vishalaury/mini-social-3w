Mini Social App (3W Internship Project)

A mini social media web application where users can sign up, log in, create posts with text or images, and interact through likes and comments.

Developer: Vishal Maurya
Email: mauryavishal9919@gmail.com

Tech Stack

Frontend: React.js

Backend: Node.js and Express.js

Database: MongoDB Atlas

Deployment:

Frontend: Vercel

Backend: Render

Features

User authentication with signup and login functionality

Create posts with text, image, or both

Public feed displaying posts from all users

Like and comment functionality for each post

Responsive and simple user interface

Deployment Links

Frontend: https://mini-social-3w-cv6o.vercel.app

Backend: https://mini-social-3w-4.onrender.com

Database: MongoDB Atlas

Folder Structure

/frontend → React application files

/backend → Express server and MongoDB configuration

/backend/models → Database schemas for User and Post

/backend/routes → API routes for authentication and posts

How to Run Locally

Clone the repository

git clone https://github.com/Vishalaury/mini-social-3w.git


Go to the backend folder and install dependencies

cd mini-social-backend
npm install


Create a .env file inside the backend folder and add your environment variables

PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key


Start the backend server

npm start


Go to the frontend folder and install dependencies

cd ../mini-social-frontend
npm install


Start the frontend application

npm start


Then open http://localhost:3000
 in your browser.
